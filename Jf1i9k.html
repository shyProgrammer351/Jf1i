문제

아래 설명된 의미에 해당하는 용어를 쓰세요 
 switch문은 switch( A ) { case B : C ... default : D }로 구성되며, 정수값 A와 B가 같으면 명령문 C 등을 실행  

 정수형 변수 score를 10으로 나누어 100점은 10, 90이상은 9, 80이상은 8, 70이상은 7, 나머지로 구분  

 case는 switch( A ) { case B : C ...}에서 사용되며, 변수 A가 정수값 B와 같으면 C를 수행하고 아니면 나머지 조건을 비교  

 switch( A ) { case B : C ... default : D }에서 정수값 A와 같은 case를 못 찾으면 명령문 D를 실행
