문제

아래 설명된 의미에 해당하는 용어를 쓰세요 
 if ( A ){ B }에서 조건식 A가 참이면 명령문 B를 실행  

 비교연산자로 A가 B보다 작거나 같으면 참(true)을 반환하고 다르면 거짓(false)을 반환  

 if ( A ){ B } else if ( C ){ D }에서 조건식 A가 거짓이고 조건식 C가 참이면 명령문 D를 실행  

 논리연산자 AND는 조건식 A와 B가 둘 다 모두 참이면 참(true)을 반환하고, 하나라도 거짓이 있으면 거짓(false)을 반환  

 if ( A ){ B } else if ( C ){ D } else { E }에서 조건식 A와 C가 모두 거짓이면 명령문 E를 실행
