문제

아래 설명된 의미에 해당하는 용어를 쓰세요 
 전위 증가 연산자로 c = ++a는 a = a + 1 연산을 먼저 실행하고 c = a 연산을 실행  

 후위 증가 연산자로 d = b++는 d = b 연산을 먼저 실행하고 b = b + 1 연산을 실행  

 전위 감소 연산자로 c = --a는 a = a - 1 연산을 먼저 실행하고 c = a 연산을 실행  

 후위 감소 연산자로 d = b--는 d = b 연산을 먼저 실행하고 b = b - 1 연산을 실행
