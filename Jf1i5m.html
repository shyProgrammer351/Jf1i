문제

증가연산자(++)와 감소연산자(--)의 전위연산자와 후위연산자를 실습하고 그 결과를 출력하는 문제입니다 이를 해결하는 다음 프로그램의 실행상태를 확인하세요 
a = 3일 때 c = ++a 적용 후 결과 a = 4, c = 4
b = 3일 때 d = b++ 적용 후 결과 b = 4, d = 3

b = 4일 때 d = b-- 적용 후 결과 b = 3, d = 4
